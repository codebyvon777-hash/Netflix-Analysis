# Netflix Content Analysis

## Project Overview

This project analyzes the Netflix Titles dataset using Python, Pandas, and Tableau to identify trends in Netflix content.

## Questions Answered

- Which countries have the most Netflix titles?
- What are the top genres on Netflix?
- How has Netflix content changed over time?
- What ratings are most common?

## Tools Used

- Python
- Pandas
- Tableau

## Data Analysis

Using Python, I:
- Cleaned missing data
- Transformed genre information
- Grouped data to find trends
- Created CSV files for Tableau visualization

## Files

- `netflix_analysis.py` - Python analysis script
- `netflix_titles.csv` - Original dataset
- `data/` - Analysis outputs
- `tableau/` - Tableau dashboard
- `images/` - Chart screenshots
